Lowlight — это психологическая игра с повторяющейся структурой, в которой ты проживаешь депрессию изнутри. Ты управляешь девочкой, чей внутренний мир принял форму дома, медленно разрушающегося изнутри. Каждый день — ритуал: умыться, поесть, навести порядок, снова лечь. А утром всё начинается заново, только дом уже другой. 
Жанр: психологическое приключение.
Платформа: Windows.

Особенности игры:
Циклическая структура:
Каждый день идёт по одному сценарию: пробуждение, выполнение рутинных задач, эмоциональная сцена, сон и новый день.
Изменяющийся мир:
Дом меняется каждое утро. Игроку нужно искать путь заново, как будто впервые.
Визуальные метафоры:
Демонстрация эмоциональных переживаний через визуальные интерактивные образы.
Художественный стиль:
Визуальное оформление в стилистике монохромного карандашного рисунка с ручной анимацией.
